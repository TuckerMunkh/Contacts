# Contributing

## Coding Principles

- Keep HTML semantic.
- Use modern JavaScript (ES modules).
- Mobile-first design.
- Prefer reusable components over duplicated code.
- Keep performance and accessibility in mind.
- Test changes on desktop and mobile before committing.

## Commit Style

Examples:

- feat: add homepage hero
- fix: improve mobile navigation
- docs: update roadmap
- style: refine button spacing
